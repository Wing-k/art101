Challenge

Problems

Results

We created a pseudocode list for the steps to making guacamole and a pseudocode list for the steps to making a connect 4 game.

Here is the list for making a connect 4 game: 

							1. Print a 6 X 7 grid with a circle in the middle of each squares
							2. Accept user input for color
								a. Asks user if they want to be red or yellow
							3. Accept user input for move
							4. Validate user move
							5. Evaluate possible moves
							6. Choose computer move
							7. Test of win or lose condition
								a. End game if 4 of the same color circles are connected in a vertical, horizontal, or diagonal row
								b. Player of that color wins
							8. Repeat if win or lose condition is not met
